🧪 Test Google OAuth

Before testing:

  1. Make sure you have created a tenant and project in your database
  2. Enter the tenant and project slugs below
  3. Make sure your backend is running on port 3002
  4. Make sure Google Cloud Console has this redirect URI:
    http://localhost:3002/api/v1/auth/google/callback

How to create a tenant/project:

1. Use your tenant auth API to create a tenant

2. Create a project under that tenant

3. Note the slug values (not the IDs)

4. Enter them above and test
